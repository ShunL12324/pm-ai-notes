# 5.4 主流 AI 开发平台

AI 应用开发的主要平台和工具概览。

## AI 编程助手

### Claude Code (Anthropic)
- **定位**：CLI 形式的 AI 编程助手
- **特点**：Agent 模式、MCP 支持、Skills 系统
- **适用**：终端用户、复杂开发任务

### Cursor
- **定位**：AI-first 代码编辑器
- **特点**：基于 VS Code、内置 AI 补全和对话
- **适用**：日常开发、代码编写

### Windsurf (Codeium)
- **定位**：AI 代码编辑器
- **特点**：Cascade 功能、多模型支持
- **适用**：团队协作开发

### GitHub Copilot
- **定位**：代码补全助手
- **特点**：深度集成 GitHub、IDE 插件
- **适用**：已有 GitHub 工作流的团队

## AI 应用开发平台

### LangChain
- **定位**：LLM 应用开发框架
- **特点**：Chain、Agent、RAG 组件丰富
- **适用**：构建复杂 AI 应用

### LlamaIndex
- **定位**：数据索引和检索框架
- **特点**：专注 RAG 场景
- **适用**：知识库、文档问答

### Dify
- **定位**：低代码 AI 应用平台
- **特点**：可视化编排、开源可私有部署
- **适用**：快速搭建 AI 应用

### Coze (扣子)
- **定位**：字节跳动的 AI Bot 平台
- **特点**：可视化搭建、插件市场
- **适用**：国内场景、快速上线

## 模型服务平台

### OpenAI API
- **模型**：GPT-4、GPT-4o
- **特点**：生态最成熟
- **定价**：按 token 计费

### Anthropic API
- **模型**：Claude 3.5/4 系列
- **特点**：长上下文、安全性
- **定价**：按 token 计费

### 国产平台
- **通义千问**：阿里，中文优化
- **文心一言**：百度，知识增强
- **智谱 AI**：GLM 系列，开源友好
- **DeepSeek**：推理能力强，性价比高

## 选型建议

| 需求 | 推荐方案 |
|------|----------|
| 日常编程 | Cursor / Claude Code |
| 快速原型 | Dify / Coze |
| 复杂应用 | LangChain + 云服务 |
| RAG 系统 | LlamaIndex |
| 成本敏感 | DeepSeek / 通义千问 |
